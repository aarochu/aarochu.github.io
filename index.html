Projects

VeriStudent

Full stack marketplace style app for college students that makes it easy to safely buy and sell textbooks and find sublets. Students sign in with Google using a dot edu email, complete a simple profile, and post listings with images, prices, and pickup or sublet details. The frontend is built with React and Expo and deployed to Vercel, with a Supabase Postgres backend handling users, listings, messages, image storage, and real time chat between buyers and sellers.

Run Club Website

Built a responsive club website using React and TypeScript, integrated Supabase for authentication and real-time data, and implemented Resend API for automated email notifications. Designed with Tailwind CSS and deployed on GitHub Pages.
